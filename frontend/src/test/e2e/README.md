# End-to-End Tests

This directory contains Playwright E2E tests for the reasoning visualization feature.

## Running Tests

```bash
# Run all E2E tests
npm run test:e2e

# Run tests in UI mode
npm run test:e2e:ui

# Run tests in debug mode
npm run test:e2e:debug
```

## Test Structure

E2E tests will be added in later tasks according to the implementation plan:
- Full reasoning flow
- Comparison mode
- Mobile responsive behavior
- Accessibility flow
- Export functionality
